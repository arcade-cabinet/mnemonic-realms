# Master Index

> Alphabetical index of every named entity in the Mnemonic Realms game bible.
> Each entry lists its canonical ID (where applicable), a one-line description, and every doc that references it.

---

## How to Read This Index

- **Bold ID** = canonical identifier used across docs (e.g., `E-SL-01`, `MQ-03`, `W-SW-04`).
- **Doc refs** use short names: `geography` = `docs/world/geography.md`, `enemies` = `docs/design/enemies-catalog.md`, etc.
- Entries marked **(ambient)** are unnamed or generic NPCs referenced by role rather than proper name.
